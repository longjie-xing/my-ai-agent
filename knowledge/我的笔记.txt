## Python 基础

Python 是一种简单易学的编程语言，适合新手入门。
- 用缩进来表示代码块（不像 C/Java 用大括号）
- 变量不需要声明类型
- 有丰富的第三方库

## AI 相关概念

### 什么是 API？
API 就是程序和程序之间"打电话"的方式。你的程序通过 API 告诉 AI "帮我回答这个问题"，AI 通过 API 把答案返回给你。

### 什么是 Prompt？
Prompt 就是你对 AI 说的话、写的指令。好的 Prompt 能让 AI 更听话。

### 什么是 RAG？
RAG（检索增强生成）就是让 AI 在回答之前，先去你的资料库里查一下相关资料，然后再回答。这样 AI 就不会瞎编了。
